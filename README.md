# YCalendar

C'est une bibliothèque qui s'utilise en ajoutant la DLL.
Vous pouvez retrouver la DLL dans le dossier source du projet bin/Debug/.net7.0-windows/YCalendar-WPF.DLL

Pour l'utilisé il faut ajouter dans la vue :  xmlns:YC="clr-namespace:YCalendar_WPF.Views;assembly=YCalendar-WPF"
et pour l'utiliser dans la vue il faut simplement faire <YC:YCalendarControl />

La calendrier affiche en bleu les événements et en orange les jours férier

Vous allez retrouver dans le dossier documentation un diagramme de classe du composant
